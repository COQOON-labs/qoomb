### ✨ Features

- Initial project setup with NestJS API and React Web frontend
- Privacy-first hybrid encryption with AES-256-GCM and RSA-OAEP
- tRPC for end-to-end type-safe API communication
- Prisma ORM with PostgreSQL database
- Redis for session management and rate limiting
- Docker Compose development environment
- Comprehensive security hardening (CSP, rate limiting, account lockout)
- JWT authentication with refresh token rotation
